1. Установите [nvm](https://github.com/nvm-sh/nvm) или [nvm-wiondows](https://github.com/coreybutler/nvm-windows)
2. `nvm install && nvm use`
3. `corepack enable && corepack install`
4. `pnpm -v` -- нажми Y
5. `pnpm install`
